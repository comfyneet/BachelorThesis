- Install IntelliJ IDEA 2017 + Java SDK 8
- Install Visual Studio 2017 (with 'ASP.NET and web development' workload)

- Import 'pom.xml' from Source\Java with IDEA, build 'App'
- Open 'RiceDoctor.sln' from Source\C# with VS, build 'WebApp'

- Run 'App', wait for the 'INFO: Ontology is loaded.' message to be displayed in the console window
- Run 'WebApp', wait for the website to be fully loaded in the browser
